# PartyPlay

This is the repository for the Party Play Application

## Installation

1. `git clone`
2. `cd PartyPlay`
3. `yarn add`
4. `yarn start`

### Secrets

Spotify API requires a `clientId` and `clientSercet` which is storage in `secret.js` at the moment. This will be changed once the server is completed.
